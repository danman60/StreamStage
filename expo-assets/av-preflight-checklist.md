- [ ] **Browser zoom 100%**, bookmarks bar hidden, fullscreen (F11) tested with Alt-Tab.

- [ ] **Second rehearsal text** from own phone — confirms Twilio + edge function + wall are live NOW, not last night.
- [ ] Phone → Do Not Disturb after the test text.
- [ ] Clicker paired; walk the stage to test range.
- [ ] Water on the podium.

## Screen hygiene (live-demo best practice)
- Fresh browser profile or Guest window: no personal bookmarks, no autofill, no notification prompts on screen.
- Hide the dock/taskbar (auto-hide on).
- Set laptop display to never sleep (Battery + Power settings) — a demo dying at minute 14 because the lid setting is 10 min is a classic.
- Bring own HDMI adapter + spare. Never trust the venue dongle.

## Restart cheatcodes
- KB demo: `R` restarts to Beat 0 instantly. `A` toggles autoplay (avoid on stage — Space is safer).
- Decks: `Home` returns to slide 1; number + Enter jumps.
- Wall: plain browser refresh; feed rebuilds itself (server holds history).
